## 发布命令

仅在 Release PR 合并后执行；先把 `<release-commit-sha>` 替换为最终 `main` Commit。

```bash
gh release create v3.1.0 \
  --repo jiumem/forgeloop \
  --target <release-commit-sha> \
  --title "Forgeloop 3.1.0" \
  --notes-file docs/releases/3.1.0-release-notes.md \
  --latest
```

Tag 必须指向合并后的最终 `main` Commit；不得从本地候选提交或未合并分支发布。
