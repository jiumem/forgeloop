# Forgeloop 3.3.0 → 3.3.1 迁移指南

3.3.1 将 `$code-review` 断裂式替换为 `$spec-standards-review`。不提供旧 Skill、别名目录、Prompt 转发或运行时兼容分支。

## 更新调用

将：

```text
$code-review
```

替换为：

```text
$spec-standards-review
```

同时更新自定义 Prompt、Workflow、自动化和文档中的活动引用。

## 重新加载

重新安装 3.3.1 插件后，在 Codex 的 Skills 页面执行刷新或完全重启 Codex，再新建任务。已加载旧 Skill 清单的任务不会原地改名。

## 行为边界

本次发布只改变 Skill 身份和活动引用。Spec / Standards 双轴隔离、范围冻结、只读审查、证据要求以及缺少 Spec 时的 `SPEC: NOT_AVAILABLE` 行为保持不变。
