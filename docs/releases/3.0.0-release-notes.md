# Forgeloop 3.0.0

Forgeloop 3.0.0 将工程交付切换为 Tracker 驱动的 Spec/Ticket 模型，并提供 20 个边界清晰的 Codex Skills。

主要能力：

- GitHub、GitLab、Local Markdown 三套等价 Tracker 路径；
- 单 Spec、多 Spec、Blocking Graph 与严格串行 Frontier；
- 独立 Coder、Spec Reviewer、Standards Reviewer；
- 有界修复、模型升级、Branch/Integration Policy、冲突重审；
- Ticket、Spec、Initiative 三层最终验收；
- 追加式幂等 Event、并发 Claim 与崩溃恢复；
- Codex Frontmatter、Invocation Policy、UI 元数据与插件 Manifest 完整校验。

这是破坏性版本。升级前请阅读 [2.5.0 → 3.0.0 迁移指南](../migrations/2.5.0-to-3.0.0.md)。
