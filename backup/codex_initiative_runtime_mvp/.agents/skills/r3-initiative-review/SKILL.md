---
name: r3-initiative-review
description: Run the formal Initiative Review (R3) for a deliverable Initiative.
---

1. Run `scripts/prepare_bundle.py --initiative-doc <doc> --profile R3 --object-key <initiative>`.
2. Review the completion matrix, G3 evidence, and residual risk ledger.
3. Finalize with `scripts/finalize_review.py`.
