# INIT-001 Design

## Goal
Build a small but complete Codex-native initiative runtime that starts from an Initiative-level entry, delegates work through Milestones, and keeps Task as the atomic evidence unit.

## Sealed decisions
- Initiative is the only primary user entry.
- Milestone is the formal stage boundary.
- Task remains the atomic engineering and evidence unit.
- Skills are the orchestration surface.
- Repo-local Python is the deterministic kernel.
- MCP is intentionally out of scope for this MVP.
